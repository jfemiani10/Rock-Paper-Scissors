# Rock Paper Scissors

A simple browser-based Rock Paper Scissors game where the player competes against a computer opponent.

## Features
- Interactive buttons for player choices
- Randomized computer opponent
- Score tracking
- Round results displayed to the user
- Game ends when a player reaches the winning score

## Technologies Used
- HTML
- CSS
- JavaScript

## How It Works
The player selects Rock, Paper, or Scissors using buttons on the page.  
The computer randomly generates its own choice.

JavaScript compares both selections and determines the winner based on the standard rules:
- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock

Scores are updated after each round until a player wins the game.
